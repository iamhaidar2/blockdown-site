Features

The same editor works on your desktop and in the browser.

🧱

Block editor

Build pages from headings, callouts, quotes, to-dos, code, tables, columns and images. Drag to reorder. Each block type has a keyboard shortcut.

✨

AI writer Pro

Ask Claude to draft, rewrite, expand, summarize or change the tone of your text. It happens in the document, not a side chat.

✓

Grammar Pro

Get grammar and style fixes you can accept one at a time. Your formatting doesn't get touched.

⤓

Export

Send your document to Markdown, HTML, PDF or Word with one click. Colors and callouts survive the export.

🛡️

Works offline

The desktop app runs with no internet connection and no account. It does not upload your documents anywhere.

☁️

Cloud sync

Sign in on the web and your documents save to the cloud, so you can open them on another computer.

Markdown vs Blockdown

Blockdown keeps what is good about Markdown and drops the parts people get stuck on.

Plain MarkdownBlockdown
EditingYou type symbols like # and * by hand.You click a block or press a shortcut.
StructureIt is one long text file.Each part is a block you can move.
Color and calloutsPlain text only.Headings, callouts and highlights have color.
TablesPipe and dash syntax that breaks easily.A table you edit in place.
ImagesA link to a file you manage yourself.Paste an image straight in.
ExportMarkdown, plus whatever your tool adds.Markdown, HTML, PDF and Word in one click.
Learning curveYou memorize the syntax.There is nothing to memorize.
The file itselfMarkdown.Still clean Markdown underneath.

Keyboard shortcuts

Every block has one. On a Mac, use Cmd instead of Ctrl.

Insert a block

Hold Ctrl (or Cmd) and press:

  • 123 Heading 1, 2, 3
  • T Text
  • O Callout
  • Q Quote
  • L Bullet list
  • N Numbered list
  • K Checklist
  • G Table
  • J Image
  • M Columns
  • E Code
  • D Divider

Edit and format

  • CtrlB Bold
  • CtrlI Italic
  • CtrlShiftE Switch edit and view
  • Esc Leave view mode
  • CtrlV Paste an image or text
  • Enter New item in a checklist or list
  • Drag the handle to reorder a block

Four export formats

.md .html .pdf .docx

Write once and export to any of these. The layout holds up in each one.
